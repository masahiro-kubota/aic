## なぜ Bag がこんなに大きいのか

この bag は `65.6 s` しかないのに `22.4 GiB` あります。主な要因は次の通りです。

- 3 本の生 RGB image stream が約 `17 Hz`
- `/observations` が image を含んだ集約 message になっている
- `controller_manager` の statistics / introspection topic が約 `425 Hz`
- `/tf`, `/joint_states`, `/controller_state`, `/clock` も高頻度

つまり、「一旦全部録る」用途としては有効ですが、日常的な反復には重すぎます。
